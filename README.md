# trymtex

`trymtex` is a modular LaTeX package for personal document setup: fonts, layout defaults, theorem styling, utility boxes, math helpers, and a small set of optional feature modules.

## Repository layout

- `source/` — canonical authored package files
- `doc/` — manual sources
- `examples/` — usage examples
- `testfiles/` — smoke/regression tests
- `build.lua` — `l3build` configuration

This repository uses the **plain-source + l3build** model. The canonical source is `source/`; an installed `tex/latex/...` tree is generated by `l3build install` and is not committed.

## Common tasks

```sh
l3build check
l3build doc
l3build install
l3build clean
```

Wrapper targets are also available through `make`.

## Notes

- `minted` requires shell escape and Python/Pygments.
- `biblatex` is optional and disabled by default.
- Internal implementation modules live beside `trymtex.sty` and are loaded with `\InputIfFileExists`.
